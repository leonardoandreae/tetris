Detailed Description

Class representing a Tetris tetromino (tile).

The tetromino is defined by its type (I, O, T, S, Z, J, L), configuration (which is stored as a 4x4 matrix indicating filled and empty cells), position on the game board and event-driven state variables allowing it to fall or soft drop.
